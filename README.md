# 🤖 DataChat — Talk to Your Dataset

> Upload any CSV and ask questions about your data in plain English. Powered by Groq LLM.

[![Python](https://img.shields.io/badge/python-3.8+-blue.svg)](https://python.org)

---

## What it does

DataChat lets you have a conversation with your dataset. No SQL, no code — just ask questions and get instant answers powered by AI.

**Example questions:**
- "Which column has the most missing values?"
- "Should I drop the Cabin column?"
- "What's the average age of passengers?"
- "Which columns need encoding before ML training?"

---

## Tech Stack

- **Python** — Core language
- **Groq API** (llama-3.1-8b-instant) — LLM for answering questions
- **Streamlit** — Web UI
- **Pandas** — Data processing
